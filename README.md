Bank Management System (Python)
1. Objective

This program simulates basic banking operations such as:

Opening a Savings Account

Depositing money

Withdrawing money (with minimum balance rule)

Calculating Fixed Deposit maturity amount

The program is menu-driven and uses functions.

2. Information Required for Opening a Savings Account

Typical details required in a bank application form:

Customer Name

Age

Address

Mobile Number

Aadhaar / ID proof

Initial Deposit Amount

Savings Account Interest Rate:
Example: 4% per year

Minimum Balance Requirement:
Example: ₹1000

3. Fixed Deposit Interest Rates (Example)
Period	Interest Rate
1 – 2 years	6%
2 – 5 years	6.5%
More than 5 years	7%

Senior citizens: +0.5% extra interest.
